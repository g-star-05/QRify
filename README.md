# QRify - Modern QR Code Platform

QRify is a modern full-stack QR code generation platform built using:

- React + Vite
- FastAPI (Python)
- Responsive UI
- QR Code Generator
- Authentication System
- Dashboard
- QR Templates

---

# Features

- Generate QR Codes
- Download QR Codes
- Modern Templates
- Responsive Design
- Authentication Pages
- Dashboard UI
- Pricing Plans
